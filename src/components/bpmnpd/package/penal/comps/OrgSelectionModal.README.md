## Attributes

---

| 参数      | 说明                                                                               | 类型    | 可选值     | 默认值    |
| :-------- | ---------------------------------------------------------------------------------- | ------- | ---------- | --------- |
| minHeight | 最小高度                                                                           | number  |            | -         |
| v-model   | 选中的组织内容，其数据解构为`[{memberId:'xxx',type:<number>4/2/1,name:'xxx'},...]` | number  |            | `[]`      |
| disabled  | 禁用选择                                                                           | Boolean | true,false | undefined |
